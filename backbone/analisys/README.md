# Backbone Analysis Archive

Os artefatos historicos que ficavam nesta pasta foram arquivados em:

- `legacy/backbone-analisys-historical/`

Uso recomendado agora:

- gerar novos backbones por `python3 -m pipeline.main build-backbones ...`
- gerar novas analises por `python3 -m pipeline.main analyze-backbones ...`
- consultar resultados novos em `outputs/backbone/` e `outputs/analysis/`
